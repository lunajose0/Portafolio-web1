# Portfolio Personal

Un portafolio web personal interactivo creado con HTML, CSS y JavaScript vanilla.

## Características

- **Diseño Responsivo**: Se adapta a cualquier tamaño de pantalla
- **Modo Oscuro**: Cambio de tema con persistencia usando localStorage
- **Navegación SPA**: Navegación fluida entre secciones sin recargar la página
- **Animaciones Avanzadas**: Efectos de escritura, revelado y más
- **Visualizador de PDF**: Para mostrar certificados y currículum
- **Fondo Dinámico**: Formas animadas que cambian con el tema
- **Botones Sociales Flotantes**: Acceso rápido a perfiles sociales

## Secciones

1. **Inicio**: Presentación con animación de texto
2. **Sobre Mí**: Información personal, habilidades y certificados
3. **Proyectos**: Muestra de trabajos realizados
4. **Experiencia**: Línea de tiempo de experiencia laboral
5. **Contacto**: Formulario de contacto e información

## Tecnologías Utilizadas

- HTML5
- CSS3 (con variables, animaciones y diseño responsivo)
- JavaScript Vanilla (sin frameworks)
- Font Awesome para iconos
- Google Fonts

## Personalización

Para personalizar este portafolio:

1. Reemplaza los textos en `index.html` con tu información
2. Modifica los enlaces en los botones sociales y en la sección de proyectos
3. Cambia los colores editando las variables CSS en `:root`
4. Ajusta las animaciones según tus preferencias
5. Añade o quita secciones según necesites

## Uso

Simplemente abre el archivo `index.html` en cualquier navegador moderno o sube los archivos a tu servidor web.
